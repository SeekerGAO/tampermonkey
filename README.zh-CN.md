# Quick Scroll Navigation

[English](./README.md) | 简体中文

Quick Scroll Navigation 是一个轻量级的 Tampermonkey 油猴脚本，用于在网页右下角生成一个悬浮操作面板，让用户可以一键跳转到页面顶部、中部或底部。

这个脚本适合长页面浏览场景，例如文档站、论坛、博客、搜索结果页、文章详情页等，核心目标是减少重复手动滚动带来的操作成本。

## 功能特性

- 在页面右下角增加固定悬浮面板。
- 提供 `顶部`、`中部`、`底部` 三个快速跳转按钮。
- 使用平滑滚动，交互更自然。
- 兼容桌面端和移动端尺寸页面。
- 默认支持大多数 `http` 和 `https` 网站。
- 通过检测面板是否已存在，避免重复注入。

## 适用场景

- 阅读长文章后快速回到顶部。
- 浏览超长页面时快速跳转到中间位置。
- 快速抵达评论区、页脚或页面底部内容。
- 提升日常网页浏览效率。

## 实现原理

脚本会动态注入以下内容：

- 一个固定定位的悬浮面板。
- 三个绑定滚动行为的按钮。
- 一段用于布局、视觉样式和移动端适配的 CSS。

滚动逻辑基于页面的主滚动容器：

- `顶部`：滚动到 `0`
- `中部`：滚动到页面最大滚动距离的一半
- `底部`：滚动到页面最大可滚动位置

脚本通过 `document.scrollingElement || document.documentElement` 获取标准页面的主滚动节点。

## 安装方式

### 环境要求

- Chrome、Edge、Firefox 等支持油猴扩展的浏览器
- [Tampermonkey](https://www.tampermonkey.net/) 或兼容的用户脚本管理器

### 安装步骤

1. 在浏览器中安装 Tampermonkey。
2. 打开脚本文件：
   [`quick-scroll-navigation.user.js`](./quick-scroll-navigation.user.js)
3. 将脚本内容复制到 Tampermonkey 新建脚本页面，或直接导入该文件。
4. 保存并启用脚本。

## 使用说明

安装完成后：

1. 打开任意受支持的网站。
2. 在网页右下角找到悬浮按钮组。
3. 点击对应按钮：
   - `顶部`：回到页面最上方
   - `中部`：跳转到页面中间位置
   - `底部`：跳转到页面最下方

## 脚本元信息

当前脚本使用以下元信息配置：

- `@match http://*/*`
- `@match https://*/*`
- `@run-at document-idle`
- `@grant none`

这意味着脚本默认会在绝大多数常规网页中生效，同时无需依赖额外权限或构建流程。

## 兼容性说明

- 对标准文档滚动页面支持较好。
- 某些高度定制化的网站会使用独立滚动容器而不是页面主文档滚动，这一版本暂未专门处理。
- 某些页面已有悬浮工具栏时，右下角面板可能发生视觉重叠。

## 当前限制

- 还没有提供收起或隐藏功能。
- 还没有按站点启用或禁用的配置能力。
- 暂未支持嵌套自定义滚动容器识别。
- 暂未提供键盘快捷键。

## 可扩展方向

- 增加收起/展开按钮。
- 支持自定义按钮位置。
- 提供多语言按钮文本。
- 增加键盘快捷键。
- 兼容更多单页应用中的自定义滚动容器。
- 增加按域名配置的启用/禁用规则。

## 开发说明

这个仓库刻意保持简单，核心脚本集中在单个 `.user.js` 文件中，不依赖构建工具、包管理器或外部运行时。

如果需要修改功能：

1. 编辑 [`quick-scroll-navigation.user.js`](./quick-scroll-navigation.user.js)
2. 在 Tampermonkey 中重新保存或重新导入脚本
3. 刷新目标网页并直接验证效果
